# `_and_`
The primitive for logical AND. It performs element-wise logical AND on boolean arrays, or scalar logical AND if both operands are scalars.

>**Details:** SaC only supports logical operations on actual boolean types.

### Variants In `prf.def`
- `_and_SxS_`
- `_and_SxV_`
- `_and_VxS_`
- `_and_VxV_`
