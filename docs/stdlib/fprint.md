# `fprint`
Prints values to an explicit stream.

### Common Signatures
```sac
void fprint(TermFile &stream, <a>[d:shp] arr)
void fprint(TermFile &stream, string fmt, <a>[d:shp] arr)
void fprint(File &stream, <a>[d:shp] arr)
void fprint(File &stream, string fmt, <a>[d:shp] arr)
```

### Description
Array-oriented stream output function from `ArrayIO.xsac`.
