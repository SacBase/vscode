# `tos`
Converts values to string.

### Common Signatures
```sac
string tos(int x)
string tos(float x)
string tos(double x)
string tos(bool x)
string tos(<supported>[+] x)
string tos(<supported> x, int indent)
```

### Description
String conversion functions are defined in `String.sac` for scalars and structured array pretty-printing.
