# `toi`
Converts to integer.

### Common Signatures
```sac
int toi(<numeric> x)
int[d:shp] toi(<numeric>[d:shp] x)
int toi(string str)
int, int toi(complex c)
```

### Description
`toi` is overloaded across StdLib modules for scalar/array numeric conversion, string parsing, and complex decomposition.
