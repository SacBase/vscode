# SaC Diagnostics Playbook

Use this skill when diagnostics are noisy, missing location data, or hard to act on.

- Prefer deterministic parser fixes before adding heuristics.
- Preserve original compiler context in related information.
- Keep user-visible messages concise and actionable.
- Validate behavior in expanded, smart, and hybrid presentation modes.
